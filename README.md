### Installation

1. Clone the repo

```sh
git clone https://github.com/AgungWicaksonoPutro/covid19.git
```

2. Install NPM packages

```sh
npm install || yarn install
```

3. Compiles and hot-reloads for development
```sh
npm run serve || yarn serve
```
